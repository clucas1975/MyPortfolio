        "After being in Education and the hardware side of IT for several years, I had decided to look into the software side of IT initially as a Software Quality Assurance Engineer and Tester. As of 2021, I'm expanding my software knowledge in becoming a Software Developer."

        What is a Software Developer?

        "A Software Developer is a person who programs or codes, builds, designs, installs and tests a software system from scratch!"

        What is the difference between a Software Quality Assurance Engineer and a Software Tester?

        "A Software Quality Assurance Engineer oversees every aspect of software development. This person designs based on the requirements a criteria to ensure that the software has met quality standards. While testing a software is a major responsibility of the Software Quality Assurance Engineer, it's not its only responsibility. A Software Tester simply tests the software. Mostly the Tester's intent is to purposely break code or find bugs in the software for the Developers to resolve and fix."
